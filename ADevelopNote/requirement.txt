Sail 取代傳統的 php, composer, npm 指令，寫法如下：
    ./vendor/bin/sail artisan migrate      # 跑資料庫 migration
    ./vendor/bin/sail composer require guzzlehttp/guzzle  # 安裝套件
    ./vendor/bin/sail npm install          # 前端套件
    ./vendor/bin/sail npm run dev          # 建構前端資源
為了簡化，你可以在 .bashrc / .zshrc 加 alias：
    alias sail='[ -f sail ] && bash sail || bash vendor/bin/sail'
